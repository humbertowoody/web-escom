Diferencias entre la Web 1.0, 2.0, 3.0 y 4.0

Web 1.0

La web 1.0 es la forma más básica que existe de navegadores de solo texto. Se basa en la Sociedad de la Información, en medios de entretenimiento y consumo pasivo (medios tradicionales, radio, TV, email). Las páginas web son estáticas y con poca interacción con el usuario. Esta web primitiva es estática, centralizada, secuencial, de solo lectura, y es no interactiva.

Web 2.0

El término web 2.0 fue acuñado por O’Reilly en 2004 para referirse a una segunda generación de tecnología web basada en comunidades de usuarios y una gama especial de servicios, como las redes sociales, los blogs, los wikis, los chat, foros, álbumes de fotografía, presentaciones en red, etc., que fomentan la colaboración y el intercambio ágil de información entre los usuarios de una comunidad o red social. La web 2 posibilita la conexión de personas con personas –redes sociales, wikis, colaboración, con posibilidad de compartir--.

Web 3.0

Conocida como la “web semántica” porque utiliza de forma más eficiente los datos: “data web”. Es inter-operativa y el usuario tiene el control para hacer los cambios que desee modificando directamente las bases de datos. Está asociada al concepto de personalización.

Web 4.0

En el 2016 empezó la web 4.0, que es el próximo gran avance y se centrará en ofrecer un comportamiento más inteligente, más predictivo, de modo que podamos, con sólo realizar una afirmación o petición, poner en marcha un conjunto de acciones que tendrán como resultando aquello que pedimos o decimos.

Cuadro comparativo

Web Características
Web 1.0
  • Sólo HTML
  • No se puede hacer una página dinámica
  • No se puede hacer una página de contacto
  • No se puede hacer una página de error
  • No se puede hacer una página de búsqueda
  • No se puede hacer una página de registro
  • No se puede hacer una página de inicio
Web 2.0
  • HTML + CSS
  • Se puede hacer una página dinámica
  • Se puede hacer una página de contacto
  • Se puede hacer una página de error
  • Se puede hacer una página de búsqueda
  • Se puede hacer una página de registro
  • Se puede hacer una página de inicio
Web 3.0
  • HTML + CSS + JavaScript
  • Se puede hacer una página dinámica
  • Se puede hacer una página de contacto
  • Se puede hacer una página de error
  • Se puede hacer una página de búsqueda
  • Se puede hacer una página de registro
  • Se puede hacer una página de inicio
Web 4.0
  • HTML + CSS + JavaScript + PHP
  • Se puede hacer una página dinámica
  • Se puede hacer una página de contacto
  • Se puede hacer una página de error
  • Se puede hacer una página de búsqueda
  • Se puede hacer una página de registro
  • Se puede hacer una página de inicio
